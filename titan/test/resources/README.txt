This folder contains resources for Setup class unit tests.
\test_input\ folder contains test data
\test_model\ folder contains the 'm_a_g_i_c___power_scheduling_problem' GAMS model
\test_output_for_magic_invest\ folder contains output files of Magic Investments. Investments.gdx is required input
    file for Magic Operation
\test_tool\ contains a test tool definition file for testing for example the ToolInstance class
